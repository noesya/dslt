## Architecture

### Système cœur :

Les grandes règles et utilitaires CSS : 
- Grilles
- Typographique
- Couleurs sémantiques
- Gestion des icônes
- Breakpoints
- Espacements (4px)
- Composants

### Thème defaut :

Thème par défaut Spyri (Heidi, Kometa, Geneva).
